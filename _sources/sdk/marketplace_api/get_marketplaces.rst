================================================
Get all marketplaces
================================================

.. tabs::

  .. tab:: REST

    .. code-block:: sh

        API Endpoint: <IMPROMPT_SERVER_URL>/marketplaces
        Method: GET
        X-Api-Key: API_KEY


  .. tab:: CURL

    .. code-block:: python

      curl --location --request GET '<IMPROMPT_SERVER_URL>/marketplaces' \
        --header 'Content-Type: application/json' \
        --header 'Authorization: API_KEY' \
        '
